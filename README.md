# GlassPomodoro
一个纯前端、零依赖的番茄钟网页应用

## 功能特性

### 番茄钟
- 三种模式：**工作 / 短休 / 长休**
- **自动切换工作/休息**（可开关）
- **结束提示音**（可开关）
- **长休周期**：每 **N** 轮进入一次长休（可设置）
- 统计面板：已完成工作段/休息段、当前模式、剩余时间

### 背景
- **多背景源容错**：一个源失败会自动切换到下一个
- **手动换背景**
- **自动换背景**：按“分钟”设置间隔（可开关）

### UI / 交互（重点）
- **“更多”二级菜单**
  - 桌面端：根层级 `position: fixed` 浮层，自动贴合按钮位置
  - 移动端：底部抽屉（bottom sheet），不影响布局
- **“时长设置”二级弹窗**
  - “更多”里只提供入口按钮
  - 点击后弹出独立设置面板（移动端同为抽屉式）
- **沉浸模式（Cinematic）**
  - 长时间不交互会自动触发（15 秒）：界面主体渐隐，仅保留缩小倒计时
  - 支持手动按钮与快捷键 `I` 触发/退出
  - 缩小倒计时支持自由拖动位置

### 动画与主题体验
- 更自然的非线性过渡（`cubic-bezier`）
- 背景呼吸感缩放、卡片轻微浮动、面板开合柔和动画
- 新增黑夜模式（快捷键 `D`）
- 支持自定义背景透明度

### 数据存储
- 使用 `localStorage` 保存配置与部分状态：
  - 工作/短休/长休时长、长休周期
  - 自动切换、提示音
  - 自动换背景开关与间隔
  - 背景源索引、统计数据等

## 代码结构
- `index.html`：页面结构与语义标记
- `styles.css`：样式与动画
- `app.js`：业务逻辑与事件管理
